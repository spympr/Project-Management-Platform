### Πως δουλεύει
Η σκοπός της σελίδας Πως δουλεύει είναι να εξηγήσει αναλυτικά πως μπορούν μια ομάδα να εκμεταλλευτεί τα Scrum εργαλεία της πλατφόρμας, ούτως ώστε να βελτιωθούν οι διεργασίες κάθε Sprint.
