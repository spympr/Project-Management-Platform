## Στόχος του συστήματος 
Ανάπτυξη μιας διαδικτυακής συνεργατικής πλατφόρμας διαχείρισης έργων λογισμικού. Η πλατφόρμα θα επιτρέπει σε ομάδες να υλοποιούν πολλά έργα λογισμικού με βάση τη μεθοδολογία Scrum, υποστηρίζοντας τις βασικές έννοιες των Backlogs, Sprints και User Stories καθώς και τις σχετικές αναφορές. Χρήση του μοντέλου «Λογισμικό ως Υπηρεσία» (Software as a Service – SaaS).

## Εμβέλεια συστήματος
Το σύστημα θα προορίζεται για άτομα και ομάδες που θέλουν να οργανώσουν την διαχείριση ενός έργου με βάση την Scrum μεθοδολογία. Το σύστημα παρέχει τις βασικές λειτουργίες διαχείρισης έργου με σαφήνεια και καθαρότητα, ώστε οι χρήστες να επικεντρώνονται στην γρήγορη και αποτελεσματική αποπεράτωση του έργου τους.


## Απαιτήσεις
Εδώ περιγράφονται οι λειτουργικές απαιτήσεις του συστήματος. Αυτές είναι ταξινομημένες σε ομάδες με βάση την ρόλο τους στην εφαρμογή. Η κάθε ομάδα έπειτα κατηγοριοποιείται σε σελίδες οι οποίες διαθέτουν τις εξειδικευμένες λειτουργίες της εφαρμογής. Κάθε απαίτηση σελίδας έχει έναν σύνδεσμο ο οποίος μεταβαίνει στον πίνακα προδιαγραφών της απαίτησης. Εκεί εξηγείται αναλυτικότερα καθώς και ο τρόπος εκτέλεσής της.

#### Σελίδα Projects
##### Αναλυτική προεπισκόπηση όλων των projects
| Απαίτηση           | Αναλυτική προεπισκόπηση όλων των projects                    |
| ------------------ | ------------------------------------------------------------ |
| **Σελίδα**         | Projects                                                     |
| **Προδιαγραφή**    | Μόλις συνδεθεί ο χρήστης θα μπορεί από το κουμπί Projects στην πάνω μπάρα να μεταβεί στην σελίδα των Projects. Εκεί θα μπορεί να δει αναλυτικά σε λίστα όλα τα Projects που συμμετέχει ενεργά και μη. Πατώντας πάνω σε ένα Project από την λίστα θα μπορεί να μεταβεί σε αυτό.    |

##### Δυνατότητα για δημιουργία νέου project
| Απαίτηση           | Δυνατότητα για δημιουργία νέου project                       |
| ------------------ | ------------------------------------------------------------ |
| **Σελίδα**         | Projects                                                     |
| **Προδιαγραφή**    | Στην σελίδα των Projects θα υπάρχει ένα κουμπί Δημιουργία Project. Πατώντας το θα ανοίγει ένα παράθυρο. Σε αυτό το παράθυρο ο χρήστης θα μπορεί να ορίσει το όναμα του Project και να δώσει μια σύντομη περιγραφή του. Έπειτα θα μπορεί να προσθέσει άτομα στο Project με 2 τρόπους. Ο ένας θα είναι να προσκαλέσει μια ολόκληρη ομάδα στο Project και ο δεύτερος θα είναι να προσθέτει μέλη ξεχωριστά ένα ένα με τα email τους. Θα μπορεί να προσθέσει ομάδα και μέλη μαζί το ένα μετά το άλλο. Τέλος άμα είναι premium ο λογαριασμός του χρήστη θα μπορεί να επιλέξει να κάνει το Project Premium. Αυτή η αναβάθμιση θα μπορεί να γίνεται μόνο κατά την δημιουργία του Project.    |
